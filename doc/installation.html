Installing OpenMEE

  1. To install OpenMEE, first download the program from:
    http://www.cebm.brown.edu/open_mee/download

    2. Make sure to download the correct version of the program for your system. As of this writing, we currently support Mac OSX Mountain Lion and Windows 7&8 (64-bit)
  2. Save the zip file somewhere convenient and extract it (just double-clicking should do on most systems).
  3. Make sure that the location from which you run the program is somewhere where you have write privileges (somewhere in your user path i.e. Somewhere after
    /Users/yourname/ on Mac or
    C:\Users\yourname\ on windows).

    The program needs to be able to write to its own directory.
    4. Do not put the program in the Program Files folder on a PC or Applications folder on a Mac!

Starting OpenMEE

Launch the program by clicking on LaunchOpenMEE.exe in the case of Windows or, on Mac, the OpenMEE app. This will open the main spreadsheet. Sometimes this can take a few seconds.
