Introducing OpenMEE

OpenMEE is open-source software for performing meta-analysis suited to the need of ecologist and evolutionary biologists.

Installing OpenMEE

  1. To install OpenMEE, download the program from:
    http://www.cebm.brown.edu/open_mee/download
  2. Save the zip file somewhere convenient and extract it.

Starting OpenMEE

Launch the program by clicking on LaunchOpenMEE.exe in the case of Windows or the OpenMEE app in the case of Macintosh. This opens the main spreadsheet as shown below.

The OpenMEE Spreadsheet

From this spreadsheet you can enter your data by:
  1. Entering the data manually by typing it in to the spreadsheet.
  2. Opening a previously saved dataset (.ome) file from the file menu
  3. Importing a CSV file (with or without column headers)

The OpenMEE spreadsheet is very flexible. It has an unlimited number of rows and columns. The spreadsheets starts off with a fixed (large) size. Calculating an Effect Size

Before we can run an analysis, we need an effect size and variance. We can get this in one of two ways:
  1. Entering an effect size and variance manually (e.g. when raw data is not available)
  2. Calculating an Effect Size from the raw data

We will examine each option in turn

Entering an effect size and variance manually

To do this, we simply create two new continuous variables using the method we described earlier. Let's call them "my effect size" and "my variance". In order to use these variables as effects in a meta analysis we need to tell OpenMEE that these are effects. Right click on the header for "my effect size" and choose 'Change format -----> Trans. Effect'. Likewise, choose 'Change Format ----> Trans. Var' for "my variance" as shown in the Figure below.

Once we have finished changing the format of these two variables, we have the screen shown below

We show now take an aside to discuss transformed-scale vs. raw-scale. In order to perform a meta-analysis, we need an effect size and a single measure of dispersion for each study. For example, if our metric is "Odds ratio" we need the effect size on the transformed (log) scale since confidence intervals for odds ratios are generally not symmetric in the raw (untransformed) scale. It just so happens, however, that OpenMEE is capable of transforming an effect size and confidence interval or effect size and variance from one scale to the other. We will see how to do this soon.

Calculating an Effect Size from the raw data

In order to calculate an effect size from raw data, click the 'Calculate Effect Size' button on the toolbar or select 'Calculate Effect Size' from the analysis menu. This will show the wizard in Figure 10. First, you have to select a data type and a metric.

Then, you tell OpenMEE the locations of the necessary variables as show in Figure 11. You will be given the option to link the raw data columns to the resulting effect size columns (selected by default). This makes it so that when you change a value in the raw data, the effect column result will update automatically.

In Figure 12, we see the calculated effect size and variance in the 'transformed scale' (lnOR and Var(ln OR)). Also, note on the bottom of the window, we see a red bar. This provides a visual cue that the four raw data columns cont. #events, #cont #noevents, etc are linked to the effect and variance columns.

Finally in Figure 13, we see the result of transforming the effect size. Three new columns have been added containing the 'raw scale' effect size and upper and lower confidence bounds given the current confidence level shown in the toolbar. Changing the confidence level in the toolbar will cause the confidence interval to recalculate.

Running Analyses

All analyses are performed similiarly. You can choose to perform an analysis by clicking its icon in the toolbar or by selecting it from the Analysis menu. More exotic analyses such as bootstrapped meta regression, meta-regression-based conditional means and, boostrapped meta regression-based conditional means are only accessible from the menu bar.

Since all the analyses are similiar, I will describe the pages you see on the wizards for each analysis.

Standard Meta Analysis

  1. Data Type & Metric
  2. Data Location
  3. Refine studies/categories/exclude studies with specific missing data
  4. Analysis Method and Parameters
  5. Summary Page

Cumulative Meta Analysis

Leave One Out Meta Analysis

The options for Cumulative Meta Analaysis and Leave-One-Out meta analysis are the same as for Standard Meta Analysis

Subgroup Meta Analysis

The options for subgroup meta analysis are the same as for standard meta analysis except for a categorical variable must be selected (SHOW PICTURE). For example, if the selected variable is Gender, the subgroup analyses will be run on the Male and Female subgroups.

Meta Regression

The wizard for subgroup meta analysis are the same as for standard meta analysis aside two additional pages.

  1. On the 'Select covariates for regression page', you select the covariates for regression(big surprise). On this same page you choose whether you want to use a random or fixed effects model and the confidence level to use in the analysis.
  2. Reference Values - If you select categorical covariate(s) you can opt to choose which value will serve as the reference value (for the intercept)

Boostrapped Meta Analysis

The options for boostrapped meta-analysis are the same as those for standard meta analysis aside from the fact that on the boostrap parameters page you can choose choose various parameters for the bootstrap including the # of bootstrap replictes to perform. When running bootstrap operations in OpenMEE, the program tries to generate results using the selected # of bootstrap replicates. So, if one particular replicate attempt fails, another is tried, and so on until 5 times the selecte # of bootstrap replicates have been attempted, at which point an error message is returned
